# Anexos de soporte (opcional)

En este directorio puedes añadir como anexo:
- Reglas de iptables
- Reglas de snort
- Configuraciones de tripwire
- Plantillas de OpenVPN (sin claves/secretos)
- Scripts auxiliares (si aplica)

Recomendación: no incluir credenciales, claves privadas o certificados reales.
